Musk Rocket Simulator

Start the game by pressing R

Controls:

MUTE: press M key

PAUSE: press P key

THRUST: hold W key

ROTATE COUNTERCLOCKWISE: hold A key

ROTATE CLOCKWISE: hold D key

RELOAD PAGE (after death): press R key
